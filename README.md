# Projeto de Arquitetura de Software

## Passos pipenv
- dnf install python-pip
- pip help
- sudo dnf install pipenv

## Na pasta do projeto "app"
- pipenv install 
- pipenv shell
- pipenv run python app.py


### Install Flask
```bash 
pip install Flask
```
### Para conectar ao SQL
```bash
pip install pyodbc
```
